# word-to-pdf-py
Bu aracı kendi bilgisayarınızda çalıştırmak için şu adımları izleyin:

1. Gereksinimleri Yükleyin:
Terminalinizi açın ve şu komutu yazarak gerekli kütüphaneyi kurun:
pip install docx2pdf

2. Dosyaları Hazırlayın:
Dönüştürmek istediğiniz .docx dosyasını, main.py ile aynı klasöre koyun.

3. Kodu Çalıştırın:
Terminalde şu komutu girin:
python main.py

4. Dosya Adını Girin:
Ekranda dosya adı sorulduğunda, Word dosyanızın tam adını (örneğin: belge.docx) yazıp Enter'a basın.
